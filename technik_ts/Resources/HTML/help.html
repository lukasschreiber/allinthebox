Barcodescanner

Mit dem Barcodescanner können die Items schnell gefunden werden. Es wird in der Tabelle automatisch zum gescannten Item gescrollt und dessen Zeile ausgewählt. In dem Seitenfenster rechts werden alle Informationen zu diesem Item angezeigt.


Hinzufügen:

Wenn der gescannte Barcode noch nicht inventarisiert wurde, erscheint ein Fenster, in dem man direkt Name des Items und die erste Zeile der Bemerkungen notieren kann und das Item kann hinzugefügt werden.


Einstellungen Barcodescanner:

Diese Einstellungen müssen nur vor Erstnutzung und bei Barcodescannerwechsel geändert werden.


Minimum Länge:

Dieser Wert gibt die minimale Anzahl an Zeichen im Barcode an.


Suffix Minimum:

Dies gibt an wie oft der Suffix vom Barcodescanner an den PC gesendet wird.


Suffix:

Der Suffix, der am Barcodescanner eingestellt wurde, muss in dem Feld neben „Suffix“ ausgewählt werden.

Diese Einstellungen können nur vom Admin geändert werden.


Farben auswählen

Um eine Farbe zu definiere, klickt man in den Einstellungen auf das in der aktuellen Farbe eingefärbte Rechteck daneben. Die Farbe kann dann aus den „Basic Colors“ ausgewählt werden, indem man auf die gewünschte Farbe klickt oder man kann unter „Define Custom Color“ eine eigene Farbe definieren. Dazu wählt man mit der Maus in dem großen Feld eine Farbe aus und kann auf der Skala rechts die Helligkeit verstellen. Die ausgewählte Farbe wird in dem Feld „Color|Solid“ angezeigt. Farben können auch über das HSL oder RGB Schema definiert werden. Danach muss man die Auswahl über „Add to Custom Colors“ bestätigen.


Fenstereinstellungen

Hier kann in den Einstellungen ausgewählt werden, in welcher Ebene sich neue Fenster öffnen. Zudem kann man auswählen, dass jedes Fenster nur einmal geöffnet werden kann.


Fußzeile

Neben dem Copyright steht durch Zahlen angegeben das wievielte Item gerade ausgewählt ist und wie viele Items insgesamt inventarisiert wurden.


Einstellungen:

In der rechten Ecke befindet sich ein Zahnradsymbol, bei dem man zu den Einstellungen gelangen kann.


HotKeys

In den Einstellungen kann man unter diesem Begriff Tastenkombinationen für Kurzbefehle individuell festlegen. Um Hotkeys festzulegen, muss man auf das Feld neben dem Befehl klicken und die Tasten/ Tastenkombinationen drücken, die man festlegen will.


Version:

In einem Fenster wird angezeigt, welche Version vom Programm installiert ist.


Zurücksetzen:

Unter „Hotkeys zurücksetzen“ können alle Hotkeys auf den Standard zurückgesetzt werden.

Diese Einstellungen können nur vom Admin geändert werden.

Login

Anmelden mit Passwort:

Beim Login muss man Nutzername und Passwort eingeben. Beim Nutzername ist die Groß und Kleinschreibung egal, beim Passwort nicht! Beide Felder müssen ausgefüllt sein. Die Eingabe kann auf der „Einloggen“ Schaltfläche oder mit Enter bestätigt werden. Das Passwort kann durch fahren der Maus über das Auge sichtbar gemacht werden.


Anmelden mit USB Authentifikation:

Ein zuvor hinzugefügter USB-Stick kann vor dem Programmstart oder während des Logins einstecken werden und kann nach der Authentifizierung wieder entfernt werden. Wenn der Stick umbenannt wurde muss er neu hinzugefügt werden!

Verlassen:

Durch Klicken auf das „x“ oder die „Verlassen“ Schaltfläche wird das Programm vor dem „richtigen“ Öffnen geschlossen.

Das nicht korrekte Laden der Anmeldeoberfläche kann damit zusammenhängen, dass die Internetverbindung vorhanden, aber sehr schwach ist oder damit, dass die Log.xml Datei zu groß ist.


MultiAuswahlTool

Hiermit können mehrere Items auf einmal ausgeliehen oder zurückgegeben werden. Dazu kann man den Barcode der einzelnen Items scannen und nach jedem Barcode Pfeil-nach-unten drücken. Der Name des gescannten Items wird angezeigt, sobald die Pfeiltaste gedrückt wurde. Danach kann man über die Buttons links alle Items der Liste ausleihen bzw. zurückgeben. Ein kleiner Stern * am Anfang der Zeile zeigt an, dass diese als nächstes bearbeitet werden soll, ein Pfeil zeigt an welche Zeile ausgewählt ist.


Datei speichern:

Über die Abfrage beim Ausleihen oder über „Als Dokument speichern“ kann man die Liste als Word oder Exceldatei speichern, wobei der Vorgang bei Exceldateien etwas länger dauert. In dem Dokument steht das Datum, an dem die Liste erstellt wurde und der Name des Nutzers, der gerade angemeldet war. In der Tabelle stehen Name, Barcode und Bemerkungen zum Item und es gibt eine Spalte in der abgehakt werden kann, dass man das Item wieder eingepackt hat.


Zurücksetzen:

Über diesem Button werden alle Items aus der Liste entfernt.


Nutzermanager

Hier kann der Admin die Nutzer verwalten. In der rechten Spalte stehen die Namen aller gespeicherten Nutzer. Wenn ein Name ausgewählt wird, werden alle Informationen über den Nutzer angezeigt. Ein neues Passwort kann in dem entsprechenden Feld festgelegt werden. Der Name kann nicht geändert werden. Es muss stattdessen ein neuer Nutzer hinzugefügt werden.


USB-Stick:

Es kann ein USB- Stick für das Automatische Anmelden ausgewählt werden. Dazu muss man den Stick in den PC stecken und dann den Nutzermanager öffnen. Der Stick kann dann im dritten Feld von oben ausgewählt werden. Beim Starten des Programms kann man sich dann Anmelden mit USB-Authentifikation. Die Änderungen müssen gespeichert werden.


Admin/Nutzer:

Es kann ausgewählt werden, ob der Nutzer Admin oder normaler Nutzer ist. Die Änderungen müssen gespeichert werden.


Löschen:

Unter Löschen wird der Nutzer unwiderruflich gelöscht. Dies kann durch ein Backup rückgängig gemacht werden.


Neuer Nutzer:

Hiermit kann ein neuer Nutzer hinzugefügt werden. Der Name erscheint in der Liste der Nutzer und kann dort ausgewählt und bearbeitet werden.

Diese Einstellungen können nur vom Admin geändert werden.


Regalmanager


Schnellauswahlbuttons

Die Schnellauswahlbuttons befinden sich unterhalb der Tabelle.


Aktualisieren:

Wenn manuell die Data.xml Datei bearbeitet wurde, muss die Datei aktualisiert werden.


Hinzufügen:

Es erscheint ein Fenster, in dem man Barcode, Name und die erste Zeile der Bemerkungen notieren kann. Der Barcode kann mit dem Barcodescanner gescannt oder manuell eingegeben werden. Nach dem Hinzufügen wird das Item direkt ausgewählt und das Seitenfenster zeigt alle Informationen zum Item, die nun bearbeitet und ergänzt werden können.


Löschen:

Die ausgewählte Zeile wird unwiderruflich gelöscht, nachdem dies nochmals bestätigt wurde. Durch ein Backup kann dies rückgängig gemacht werden.


Ausleihen/ Zurückgeben:

Der Status des Ausgewählten Objekts wird geändert.
Entliehen → Verfügbar
Verfügbar → Entliehen
Die Items werden immer auf den gerade eingeloggten Nutzer entliehen.
Mehrere Items können über das MuliAuswahlTool ausgeliehen werden.


Seitenfenster (rechts)

Hier befinden sich alle Informationen über das gerade aufgerufene Item.


Barcode:

Unter dem Namen steht der Barcode, der nicht verändert werden kann. Ist dies nötig, muss das Item neu inventarisiert werden.


Name:

In dem Eingabefeld darunter kann der Name, unter dem das Item gefunden werden kann, geändert werden. Die Änderung muss über den Speicherbutton gespeichert werden.


Ausleihen/ Zurückgeben:

Durch Anklicken der kreisförmigen Buttons kann der Status des Items geändert werden. Wenn es entliehen ist, steht dort welcher Nutzer dieses gerade entliehen hat.


Bemerkung:

Hier können Informationen über das Item gespeichert werden. Dabei wird nur die erste Zeile in der Tabelle angezeigt und nur diese kann über die Suchfunktion gefunden werden. Ein „?“ zeigt in der Tabelle an, dass noch weitere Bemerkungen vorhanden sind. Mit Enter wird eine neue Zeile gestartet und diese sind auf 100 Stück begrenzt. Mit der Scrollbar oder den Pfeilen kann man zu den weiteren Zeilen gelangen, wenn mehr als 8 vorhanden sind.

Um Sachen hervorzuheben, können Sie folgende Tastenkombinationen nutzen:
Ctrl + Shift + F → dick gedruckt
Ctrl + Shift + K → kursiv
Ctrl + Shift + D → durchgestrichen
Ctrl + Shift + U → unterstrichen
Tab → · Liste
Ctrl + Shift + G → grün
Ctrl + Shift + B → blau
Ctrl + Shift + R → rot
Ctrl + Shift + S → schwarz


Speichern:

Mit diesem Button müssen Änderungen am Namen oder in den Bemerkungen gespeichert werden. Wenn ein grüner Haken vor dem Wort „Speichern“ steht, sind alle Änderungen gespeichert, wenn dort ein rotes Kreuz steht, sind die Änderungen noch nicht gespeichert.


Löschen:

Dieser Button löscht das Item mit allen Informationen unwiderruflich aus dem Programm, nachdem dies erneut bestätigt wurde. Die Daten können über das Backup wiederhergestellt werden.


Regal:

In diesem Eingabefeld kann ausgewählt werden, wo das Item aufbewahrt wird. Dies wird über den Regalmanager koordiniert.


Bewertung:

Hier können die Items mit Sternen von 1-5 bewertet werden, um den Zustand darzustellen.

Suchfunktion

Suche nach Barcode:

Man muss den Suchbegriff in das Suchfeld links eingeben und danach auf die Lupe klicken. Es wird nach einem Barcode gesucht, der den Suchbegriff enthält. Man kann in den Einstellungen „exakte Barcodesucht“ aktivieren, sodass die Suche nur noch ein Ergebnis, welches direkt ausgewählt wird, liefert.


Suche nach Name:

Wenn man den Suchbegriff in das Suchfeld eingibt, werden alle Items markiert, deren Name den Suchbegriff enthält. Die Anzahl der Treffer steht unter dem Suchfeld.


Suche nach Bemerkungen:

Wenn man den Suchbegriff in das Suchfeld eingibt, werden alle Items markiert, deren Bemerkung in der ersten Zeile den Suchbegriff enthalten. Die Anzahl der Treffer steht unter dem Suchfeld.


Suche zurücksetzen:

Sobald ein Suchbegriff eingetippt wird, erscheint ein „x“, mit welchem die Suche und alle Markierungen zurückgesetzt werden können.


Standardsuchmodus bei Start:

In den Einstellungen kann unter „Sucheinstellungen“ der „Suchstandard“ festgelegt werden. Dieser Modus wird beim Start automatisch ausgewählt.


Markierungsfarben wählen:

In den Einstellungen kann eine Farbe ausgewählt werden, in der alle gefundenen Items markiert werden. Diese kann so ausgewählt werden.


Weitere Sucheinstellungen:

Damit bereits während des Tippens des Suchbegriffes gesucht und markiert wird, kann in den Einstellungspunkt „Suchen während Tippen“ aktiviert werden. Damit automatisch zum ersten gefundenen Ergebnis gescrollt wird, muss der Einstellungspunktpunkt „Zum ersten Ergebnis scrollen“ ausgewählt werden.


Tabelle

In der Tabelle werden alle gewünschten Informationen über die einzelnen Items angezeigt. Jede Zeile zeigt die Informationen eines Items.
In den Einstellungen kann unter „Tabellenansicht“ ausgewählt werden, welche Kategorien in den Spalten angezeigt werden sollen. Die Reihenfolge der Kategorien kann geändert werden, indem man auf das Feld klickt, die Maustaste gedrückt hält und das Feld an die Stelle zieht, an der man es haben will. Die Stelle, an die das Feld verschoben wird, wird durch einen blauen Strich gekennzeichnet.


Sortieren:

Wenn man auf das Feld einer Kategorie klickt, wird die Liste nach diesem Kriterium sortiert. Bei erneutem Anklicken ist die Reihenfolge absteigend. Ob die Liste auf- oder absteigend sortiert wurde, wird durch den kleinen Pfeil an der rechten Seite des Feldes symbolisiert. Wenn an keinem der Felder dieses Symbol steht, werden die Items in der Reihenfolge, in der sie hinzugefügt wurden, gelistet.

TabellenDesign

In den Einstellungen kann man unter diesem Namen das Design der Tabelle individuell anpassen. Die Primärfarbe ist die Farbe jeder zweiten Zeile, die Sekundärfarbe die der anderen. Zudem kann auch die „Auswahlfarbe“ festgelegt werden, die das gerade ausgewählte Item hat. Wie Sie die Farben ändern können sehen Sie hier.


Schriftgröße:

Die Schriftgröße kann geändert werden, indem man den Schieberegler verschiebt oder rechts bzw. links neben den Regler auf den Balken klickt, um die Schrift um eine Einheit größer bzw. kleiner zu machen.


Entliehene Items hervorheben:

Um entliehene Items besser hervorzuheben, kann man diese zum einen kursiv anzeigen lassen, indem man entsprechendes Feld auswählt. Außerdem können entliehene Items farblich hervorgehoben werden, wenn man „Entliehenes hervorheben“ aktiviert. Zudem kann man die Farbe eines entliehenen Items, welches ausgewählt wird unter „Entliehenes Auswahlfarbe“ festlegen.


Schriftart:

Hier kann eine Schriftart ausgewählt werden.


Tipps, Sprüche oder Witze

In der Sidebar auf der linken Seite stehen im unteren Bereich tägliche Sprüche. Diese werden einer .csv Datei entnommen, also um eine comma-separated values Datei.
Die Dateien werden in einer zufälligen Reihenfolge geladen. Mit Linksklick wird die nächste geladen, mit Rechtsklick die vorherige. Dies erfolgt in Dauerschleife, wobei bei der Wiederholung die Dateien in der gleichen Reihenfolge geladen werden. Bei erneutem Öffnen der App wird eine andere Reihenfolge generiert. Wenn die .csv Datei in einem Texteditor (z.B. Notepad++) aufgerufen wird, kann diese bearbeitet werden. Jede Zeile ist ein Spruch und das Zeilenende wird durch „|“ (Alt Gr + >) markiert.
